### 本文件夹包含 Python 汉化版源文件。
* Python 版本为 3.7.3, 其余版本未测试，不保证能正常使用。
* 汉化包.py 及 \lib 目录下的文件下载后放置在安装路径的 lib 目录下。
* idlelib 文件夹中的 py 文件用于支持 IDLE 全中文化编程，下载后复制到 Python 安装路径的 lib/idlelib 目录下，
  覆盖原文件，然后正常打开 Python IDLE 即可使用。
* 运行中文程序.py 下载后放到你的工作目录下，用于支持 VS Code 环境。在 VS Code 中除安装 Python 插件外，
  还要安装 Code Runner。打开工作目录后，点击左下角的“设置”，在搜索框中输入 code runner: Custom Command，
  然后在其下方的文本框中将 echo Hello 更改为：python <你的工作目录>\运行中文程序.py $fullFileName。
  编好中文程序后，在当前文件编辑器中按 Ctrl+Alt+K 键即可运行。（当然，如果是英文编程，没必要按这三个键，还是按 F5）
* VS Code环境的中文关键词和内置函数名的颜色显示以及缩进已经可用，方法如下：
  将 vscode 文件夹中的 extension.js 文件下载后覆盖 ms-pathon 插件安装目录下的相应文件，比如在本人电脑中该文件位于：
  C:\Users\Administrator\.vscode\extensions\ms-python.python-2019.4.12954\out\client 目录中。
  将 vscode 文件夹中的 MagicPython.tmLanguage.json 文件下载后覆盖 vs code 安装目录下的相应文件，在本人电脑中该文件位于：
  C:\Users\Administrator\AppData\Local\Programs\Microsoft VS Code\resources\app\extensions\python\syntaxes 目录中。
* 推荐使用 VS Code，建议下载其简体中文包，在其插件界面中输入“chinese”就会跳出来，选中安装。
* 将 vscode 文件夹中的 pythonMain.js 文件复制到 vs code 安装目录下相应地方, 比如在本人电脑中该文件位于：
  C:\Users\Administrator\AppData\Local\Programs\Microsoft VS Code\resources\app\extensions\python\out , 然后禁用 ms-python 插件, 使用 vs   code 自带最简版本 (无智能提示功能), 就不会对中文关键词等显示恼人的警示波浪线了.
